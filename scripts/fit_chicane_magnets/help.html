Chicane Closure


This Script enables you to measure the closure of the chicane magnets and proposes a new set of ratios which can then be specified within a knob.


Procedure

  1. Switch the beam to single shot mode
  2. Open the orbit correction application and view the horizontal Ring orbit
  3. Launch the chicane closure script if it isn't already running.
    • open a terminal and login as physics
    • type: close_chicane_magnets
  4. Select the Magnets tab to view the present chicane magnet fields and the present scale factors which indicate the relative ratios used when applying field changes. Upon startup these scales are determined by the initial fields. You can edit these scales in the table if you desire to start from a different set of ratios.
  5. Go back to the Procedure tab. In stage one, you will set the initial chicane magnet fields. The actual field change should be small but big enough that you will observe a meaningful orbit difference. Warning, if the change is too big, the script will fail to yield a correct result. The default change should be fine.
  6. Press the Run button to apply the fields and move to the next stage.
  7. Circulate beam into the ring and make sure that the orbit is reasonable.
  8. Press the Run button to take a snapshot of the orbit and move to the next stage.
  9. The chicane fields will be changed to another set of trial fields. Press the Run button to apply these fields.
  10. Circulate beam into the ring and make sure that the orbit is reasonable.
  11. Press the Run button to take a snapshot of the orbit and move to the next stage. This stage may take a couple minutes before it completes as it is processing the data and fitting the measured ripple to yield a set of proposed scales.
  12. Select the Ripple tab to view the ripple that was measured.
  13. Select the Magnets tab to view the report. It should show the field errors responsible for the ripple along with the proposed scales and the measured ripple RMS and the proposed RMS. The proposed RMS Fit should be significantly smaller than the measured RMS otherwise the fit won't be good.
  14. Press the Copy Report button to copy this report to the clipboard and paste it into the logbook or a text editor or spreadsheet immediately to record the report.
  15. Select the Procedure tab and press the Run button to use the proposed scales for another experiment. Repeat the steps beginning with the setting the initial field. The measured ripple should be significantly smaller with the proposed scales.
  16. When you are satisfied with the results, you can create a new chicane knob that should be closed.
