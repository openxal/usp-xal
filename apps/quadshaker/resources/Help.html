Features and Tips:


The following order of actions is recommended:
1. Start with scan part of the application. It is the "Shaker" tab. Click the INIT button and "Make PV Logger Snapshot" before starting the scan. Use validation PV, because this will allow you to resume scan if something bad happened. Do not forget to hit the "Set PV" button before using the validation.
2. After scan, go to the "Analysis" and set the validations parameters ( it is recommended 2 for abs(Value)/err ratio , and 0.5 for minimal value of the matrix element). Then hit the "START ANALYSIS" button. If the results are not pretty you can change validation filters and use the "RE-PLOT DATA" button.
3. Go to the "Orbit Correction" tab. Hit the "Memorize Corrs. I". After that you are safe, because in any moment you can restore the initial values. Correct the H and V orbit and apply the changes.


Start the Application:
Usual start through the application launcher.
