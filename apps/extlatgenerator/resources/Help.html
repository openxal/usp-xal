ExtGen

Abstract:

This program is to generate lattice files for external programs (other than XAL on-line model) such as Trace-3d, Dynac and MAD.

Introduction Getting Started

Introduction

From top to bottom the application's frame is divided into a menu-bar, a tool bar, display panel and a status bar.


Getting Started

The first thing this application needs to do is to select an accelerator (see the above figure), then sequence(s) to to work with. These selections can be done via the Accelerator pull-down menu: select either Load Default Accelerator or Load Accelerator (will pop up a file chooser for you to pick up an accelerator file). Once you have done these selection, you will see in the display panel all the devices within the sequence(s) you just selected (see figure below).



back to top

Sequences

Next, you can select which external lattice file you would like to generate. The choices are Create T3d..., Create Dynac... and Create MAD... buttons from the tool bar.
When the external lattice file has been created successfully, a message box will appear.



back to top

You can save all the settings to a document via the File pull-down menu, then Save or Save As.... You can then open this document next time without going through most of the steps described above.
