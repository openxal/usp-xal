Knobs Application Guide


The knobs application allows users to define custom knobs and to use those knobs to control PVs simultaneously in weighted combinations. The knobs may also be grouped for convenient organization.  Note that you can save the knob definitions to a file and open the document at a later time.


Create a new Group:

  • Press the "+" button at the bottom of the "Groups" list
  • Enter a name for the Group in the dialog box and press the "OK" button
  • To delete a group, simply select it and press the back space delete key
  • The "All" group is special.  You cannot delete it.  It contains the list of all knobs defined in the document.


Create a new Knob:

  • Optionally select an accelerator (typically the default accelerator) from the "Accelerator" menu for conveniently selecting channels
  • Select the group into which you want to add the new knob
  • Press the "+" button at the bottom of the "Knobs" list
  • Select the new knob in the knobs list
  • Press the edit button on the knob view
  • To manually enter a PV, press the knob element add button at the below the element table:
    • Double click inside the PV cell and type (or drag and drop) a PV name
    • Double click inside the coefficient cell and enter a coefficient for weighting this element (amount to change PV value for one unit of knob)
  • To add a channel from the optics file definition:
    • You must have selected an accelerator from the "Accelerator" menu (typically the default accelerator)
    • From the "Accelerator Sequences" menu on the left of the dialog box, select one or more sequences
      • You may double click to select one sequence or single click on multiple sequences and press the "Select" button
    • Select one or more accelerator node types
    • Select one or more accelerator nodes
    • Select one or more PV handles
    • Select one or more Process Variables (really a reference to a Process Variable given by a node and a channel handle)
      • Note that channels selected in this way include any transforms defined in the optics file
    • Double click inside the coefficient cell and enter a coefficient for weighting this element (amount to change PV value for one unit of knob)
  • Close the dialog box when you are done
  • To delete a knob, simply select it and press the back space delete key
    • deleting a knob from a group other than "All" will only delete them from that group
    • deleting a knob from the "All" group will delete the knob from every group
  • You can drag and drop knobs onto another group to add them to that group


Generating bump knobs:

You can have the application automatically generate bump knobs for a particular accelerator sequence.

  • Select an accelerator from the "Accelerator" menu
  • Select a sequence from the "Accelerator->Sequences" menu
  • Select the group into which you want the bumps to be added
  • Select either "Make Horizontal Bumps" or "Make Vertical Bumps" from the "Generate" menu
  • In the dialog box
    • specify whether to use the design optics or synchronize with the current machine conditions
    • pick either an offset bump or an angle bump
    • specify the number of correctors to use for the bump
    • press the "Generate" button to generate the bump
    • press the "Cancel" button if you need to cancel the operation


Using the knobs:

  • Select one or more knobs from the knobs list to display their controls in the right view
  • Each knob element of a knob must have all its PVs (including set point and limits) connected in order to use the knob
  • If a warning appears in red stating "Waiting for connections..." then move the mouse pointer over the warning to view the reason in a floating tool tip
  • A knob is ready for use when it is enabled and indicated by a green "Ready" message
  • When a knob is ready, simply select the digit to vary and hit the up/down arrows to advance/retreat the digit
