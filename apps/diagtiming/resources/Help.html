·        Useful tips for using this application:

  1. For the Linac, let the app to find an optimal setting for a selected BPM first (click “Find settings for the selected BPM”).  you can then manually type in your own numbers and “Set the selected BPM”.  The “Find all” button “should” work for the Linac.
  2. “Prepare for all” will automatically figure out settings for all BPMs in that table.  All the suggested settings are in red and have not committed to the machine yet.
  3. For the Ring/RTBT BPMs, the button “Set all BPM timing” is a little bit buggy.  Use “Set the selected BPM” to set a single BPM.  The “Prepare for all” button will be based on the selected BPM to figure out the other BPMs.  Use this button, then “Set all BPM stages” for setting all BPMs.
