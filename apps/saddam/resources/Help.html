Saddam- Set Alot of Different Devices All Mutually

This is a "global lknob" application. It alows the user to select a portion of the accelerator to work with,  and change a characteristic of all like devices withing the selected accelerator sequence.

At present there is only a minimal set of "PV"s available for global knobing. If there is another one you are interested in having added, contact John Galambos (jdg@ornl.gov).


Quick Start

Quick Start

  • First select the part of the accelerator you want to work with using the Accelerator -> sequence pull down menu on top. You can create a customized accelerator sequence using the  "New Combo Sequence" option.  You must select a contiguous set of accelerator porions though.
  • A set of device types withing the selected sequence is then displayed in the "Device Type" list selection box.  Pick the device type you are intterested in.
  • Then the set of PV types for the selected device type appears in the "PV Choice" list selection. Pick the PV you want to change.
  • If the PV you selected has only certain allowed set values (e.g. the trigger type for a diagnostic):
    • these allowed values will appear in the Select Value list. Pick one. 
  • Otherwise pick an option from the "Action Choice" box:
    • Set Value = set all the PVs to this value
    • Increment value = increment the present set point value of all the PVs by this amount (adds it)
    • Multiply Value = multiply the present setpoint value by this amount
  • Fill in the value to use in the prescribed action above in the "Set Value" text field (be sure to hit return)
  • Hit "Confirm"  to get a list of the PVs you are about to affect (see text box)
  • If you are happy with everything, hit "Set'em".  If some connections cannot be made, an error message apears in the text box.
