Energy Manager Help

The energy manager provides tools for optimizing an optics. One may also use it to scale magnetic field strengths to energy and also to provide a best guess for RF phases to preserve focusing given a set of imported RF cavity amplitudes.

Common Tasks

Create a document

  1. Launch the energy manager application
  2. Specify an accelerator if one has not been set automatically (Accelerator - Load Default Accelerator) or (Accelerator - Load Accelerator)
  3. Specify a sequence (Accelerator - Sequences - The Sequence)

Load custom RF settings

  1. Create a document
  2. Parameters - Import Cavity Amplitudes...
  3. Navigate to and select the desired cavity field document (document format is cavity ID followed by the amplituded)

Select a parameter to edit

  1. Create a document
  2. Double click on a row in the parameter table for the parameter whose value you wish to edit revealing a parameter inspector
  3. Click the arrows at the bottom to navigate to the previous or next parameter listed in the table
  4. With the inspector visible you may select a new parameter simply by single clicking a row in the parameter table

Edit the custom value of a parameter

  1. Select a parameter to edit
  2. Select "Custom" as the active source if it isn't already selected
  3. Enter a new value where it says "Initial"

Make a parameter variable (or fixed)

  1. Select a parameter to edit
  2. Check the "Variable" checkbox to make it variable or deselect it to make it fixed
  3. Select multiple parameters
  4. Select (Parameters - Make Selected Variable) or (Parameters - Make Selected Fixed) as appropriate
  5. Select (Parameters - Make Disabled Cavities Fixed) to fix the phase parameters for cavities with zero amplitude

Apply best guesses to preserve longitudinal and transverse focusing

  1. Create a document
  2. Load custom RF settings or edit the initial values manually
  3. (Parameters - Apply Best Longitudinal Guess)
  4. (Parameters - Apply Best Transverse Guess)

Apply a simple energy scaling to the magnetic fields

If the sequence does not accelerate the beam then you can scale magnetic fields based on the initial energy and the target energy.
  1. Create a document
  2. Parameters - Scale Magnets For Energies
  3. Enter the initial kinetic energy and the target kinetic energy, select a source value to scale and press "Okay"

Optimize an optics

  1. Edit the custom parameter values
  2. Make at least one parameter variable
  3. (Run - Solve)
  4. Edit the objective parameters (Use Design is recommended as a starting point) and press "Run"
  5. (Parameters - Import Optimal Values) if you want to set the custom values based from the best solution found
  6. (Parameters - Upload Magnetic Fields) to upload the "Initial Values" of the magnetic fields to the control system
