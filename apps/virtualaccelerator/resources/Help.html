Virtual Accelerator Application Help

Abstract:

This application provides a virtual accelerator for software test and debugging purpose.

Getting Started

Quick Start

This application uses the standard XAL application framework. To start the application:
1. select an accelerator from the 'accelerator' pull-down menu,
2. select sequence(s) from the 'accelerator' pull down menu,
3. (optional) select a probe by pressing the 'Probe' button on the tool bar,
4. (optional) set noise level (in %) by pressing the 'Set Noise...' button,
5. start the virtual accelerator by pressing the 'Start VA' button,
6. stop a running virtual accelerator by pressing the 'Stop VA' button.
